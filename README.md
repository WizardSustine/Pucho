PUCHO

  An App for count how many cigarettes we smoke daily but also setting monthly goals.

IN-DEPTH
  To fulfill the purpose of this app i wanted that follows this requirements:
    * Have a daily counter of smokes;
    * Have a button for adding smokes;
    * Have an amount of smokes established as a monthly goal;
    * Have it automatically telling me when i can get my next smoke by dividing the amount of smokes for hours awake;
    * Have a notification when i can get my next smoke, with a shortcut to add the smoke;
    * Being able to change the goal;
    * Being able to count without a goal;
    * Being able to see a graphs of my daily progress;
    * Being able to see a list of my goals;
   USE CASE 1:
- Primary actor: User;
- Goals: Count daily smoke without a goal;
- Stakeholders: Educational technology company, students, educators, school administration, in;
- Pre-conditions: Student must be logged in, student must have access to the quiz;
- Post-conditions: Student can take the quiz and receive instant feedback on their performance
- Basic flow:
* Student logs into the educational technology platform and selects the option to take a quiz;
* System presents the student with the quiz questions
...
- Alternate path:
* Student logs into the educational technology platform and selects the option to view previous quiz results
* System presents the student with a list of previous quizzes they have taken, along with their results
